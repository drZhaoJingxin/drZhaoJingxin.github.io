概要

Windows10+github + TortoiseGit の組み合わせで使っていたところ先に「git did not exit cleanly (exit code 128)」というエラーが発生してCommitに失敗して、Fetchも出来ない状態となってしまいました。

状況

  • 前日にはGithubへのCommitを行っているが普通に使っている状況であった
  • ユーザー名・パスワード等は変更していない
  • Windows10のバージョンや强制チャットダウンも発生していない

エラーメッセージをGoogleで翻訳してみる

git did not exit cleanly (exit code 128)
gitが正常に終了しませんでした（終了コード128）

gitが正常に終了していないと発生するのかな？

しかし、Windows10を再起動しても変わらず。
